### Install via GitHub
```bash
npm install github:IsMeElyn/baileys-mod
```

Or manual clone

```bash
git clone https://github.com/IsMeElyn/baileys-mod.git
cd baileys-mod
npm install
```

Example package.json

If you want to add it directly to your package.json, you can do:
```json
{
  "name": "my-project",
  "version": "1.0.0",
  "dependencies": {
    "baileys-mod": "github:IsMeElyn/baileys-mod"
  }
}
```

Then run:
```bash
npm install
```
